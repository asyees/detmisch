NUMBERS AND OPERATIONS ON THEM

Welcome to the "Python" course. Today we will talk about numbers and operations on them in Python. Python supports various types of numbers, such as integers (int), floating-point numbers (float), and complex numbers. Let's review the basic operations that can be performed on these numbers.

a = 5
b = 3
print(a + b)

2. Floating point numbers (float): Floating point numbers have a decimal point. For example:

c = 3.14
d = 2.718

Now let's take a look at the operations that can be performed on these numbers:

  • Addition (+) and Subtraction (-): 
    result_sum = a + b
result_diff = c - d
  • Multiplication (*) and Division (/): 
    result_mult = a * c
result_div = b / d
  • Integer division (//) and remainder (%): 
    result_floor_div = a // b
result_modulo = a % b
  • Exponentiation () and **Square Root(math.sqrt()): 
    import math

result_power = a ** 2
result_sqrt = math.sqrt(b)

This is just a small part of the operations that can be performed on numbers in Python. Remember that Python also provides many built-in functions for working with numbers.
