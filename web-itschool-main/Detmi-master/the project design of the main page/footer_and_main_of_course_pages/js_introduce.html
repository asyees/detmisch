What to Expect?

This basic course covers the main features of the C++ language. Special attention is given to discussing fundamental programming principles and the compilation process. The course is not exhaustive and only covers a small part of the C++ standard, but we have tried to explain the most important features of this language.

Who Is This Course For?

This course is suitable for those who are completely new to this programming language but have a strong desire to learn it.
This course will be their guide.
