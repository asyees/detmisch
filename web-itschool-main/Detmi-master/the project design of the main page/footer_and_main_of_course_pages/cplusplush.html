C++ Course

What awaits you?

This basic course covers the main features of the C++ language. Special attention is given to discussing the fundamental principles of how programs work, as well as the process of their compilation. The course is by no means exhaustive and covers only a small part of the C++ standard, but we have tried to highlight the most important capabilities of this language.

Who is this course suitable for?

This course is suitable for those who are completely unfamiliar with this programming language but have a great desire to learn it. This course will serve as a guide for them.
